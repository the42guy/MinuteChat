MinuteChat

A tiny chat application made using basic C++ for a school project.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to Minute Chat's GitHub page

Made using C++

Beneath Minute Chat lies one of the most trusted languages - C++. You can rest assured about Minute Chat's performance and its stability.

Works over the LAN

Well, where else? A school project needs to run on the LAN. It will use a basic server-client topology, with the program files stored at a network location that can be accessed by all the users of the chat program.
